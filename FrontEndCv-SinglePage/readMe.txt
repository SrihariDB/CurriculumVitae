# Curriculum Vitae (CV) - Single Page HTML Project

This project is a single-page Curriculum Vitae (CV) created entirely using HTML. It contains essential details about the individual, including skills, education, work experience, and contact information.

## Project Structure

### Files Included:
- **`index.html`**: The main HTML file containing the CV content.
- **`style.css`**: A linked CSS file for styling the CV.
- **`Image.png`**: A favicon for the webpage.

### Sections:
1. **Header**: Displays the name of the individual.
2. **Profile**: Includes the professional title and contact address.
3. **Skills**: Highlights technical skills and areas of expertise.
4. **Education**: Lists academic qualifications.
5. **Experience**: Summarizes work experience and achievements.
6. **Footer**: Contains links to social profiles (e.g., LinkedIn, GitHub).

## Usage

### How to Run:
1. Download or clone this repository to your local machine.
2. Open the `index.html` file in any modern web browser.

### How to Edit:
1. Modify the content inside `index.html` to add or update your personal information.
2. Update the `style.css` file to customize the look and feel of your CV.
3. Replace `Image.png` with a custom favicon if needed.

## Features
- Fully responsive layout with meta tags for mobile-friendly design.
- Open Graph (OG) meta tags for better sharing on social media platforms.
- Semantic HTML structure for improved accessibility and readability.
